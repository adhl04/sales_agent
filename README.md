# Virtual Sales Agent - UAS NLP

Sistem asisten penjualan virtual berbasis AI dengan LangChain, LangGraph, dan LangSmith.

## 📋 Fitur
- 📋 Tampilkan produk
- 📦 Cek stok produk
- 🛒 Buat pesanan dengan approval
- 📊 Cek status pesanan
- 🎯 Rekomendasi produk
- 🔍 Monitoring dengan LangSmith

## 🚀 Cara Menjalankan

```bash
# Install dependencies
pip install -r requirements.txt

# Setup database
python setup_database.py

# Jalankan aplikasi
streamlit run streamlit_app.py
